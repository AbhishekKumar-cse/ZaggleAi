# CFO-OS by ZaggLeAI

CFO-OS is an AI-Native Operating System designed for enterprise financial intelligence. It provides real-time monitoring, autonomous agent orchestration, and predictive insights for modern financial leaders.

## Key Features

- **Executive Command Center**: Real-time KPI tracking and AI-generated morning briefs.
- **Spend Intelligence**: Autonomous anomaly detection for Zaggle corporate card transactions.
- **FP&A Engine**: Predictive forecasting and scenario modeling driven by Genkit and Gemini.
- **AI Agent Console**: Multi-agent orchestration for specialized financial tasks.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS & ShadCN UI
- **AI/ML**: Genkit with Google Gemini 2.5 Flash
- **Database/Auth**: Firebase & Firestore
- **Animations**: Framer Motion

## Getting Started

1. Explore the `src/ai/flows` directory to see the GenAI logic.
2. Visit `/dashboard` to see the executive overview.
3. Use the **Persistent AI Chat** at the bottom right to interact with the financial assistant.
